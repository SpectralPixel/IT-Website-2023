Pre-Classic

Features

Overview

In pre-classic, the player could move around, jump, place few blocks, and break blocks.
There was also the "human" mob, which was a steve skin running around the map randomly.

Mobs

Mob Function
Human Humans run around the map in seemingly random directions.

Blocks

  • Stone
  • Dirt
  • Cobblestone
  • Planks
  • Saplings
